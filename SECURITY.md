# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| CPAN latest   | :white_check_mark: |
| GitHub latest `master` | :white_check_mark: |
| CPAN non-latest   | :x:                |

## Reporting a Vulnerability

Please use the instructions at https://docs.github.com/en/code-security/how-tos/report-and-fix-vulnerabilities/privately-reporting-a-security-vulnerability
to report a vulnerability. Private vulnerability reporting has been enabled on this repository.
